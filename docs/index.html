What it does

Built for people who
want to do real work

Focus runs silently in the background, tracking every second of your session with zero effort on your part.

⚡
Automatic activity tracking
Polls your active app, window title, and browser URL every 3 seconds. Automatically classifies as productive, neutral, distracting, or idle — no tagging needed.
🔥
Flow state detection
Detects when you enter deep focus (25+ consecutive minutes of productive work). Shows a live flame indicator and tracks total flow time per session.
🤖
AI coaching reports
After each session, an AI model reads your activity data and writes a personalised coaching summary with specific, data-driven suggestions. Works with Ollama, Claude, or OpenAI.
👁
Vision analysis
Periodically captures your screen and uses a vision model to understand what you were actually working on — file names, documents, code identifiers. Descriptions only, screenshots discarded immediately.
📋
Day planning
Set intentions and goals each morning. Focus tracks progress against them throughout the day and suggests incomplete goals as session starters.
📈
Streaks & journey
Tracks your daily focus streak, weekly trends, top productive apps, and biggest distractions over time. Builds a picture of your actual work patterns.
AI Coaching Setup

Three ways to power
your session reports

AI coaching is optional — tracking, flow detection, and reports all work without it. When you're ready, choose whichever path fits you.

Ollama (Local) Free
Private, runs on your Mac, no API key needed
# Install Ollama
brew install ollama
ollama serve
 
# Pull models
ollama pull phi4-mini
# ^ text reports (~2.5 GB)
 
ollama pull minicpm-v:2.6
# ^ vision analysis (~5.5 GB)
Then open Settings → AI → Ollama and click Test Connection.
Claude (Anthropic) API Key
Best quality. ~$0.01 per session report.
# 1. Get your key
console.anthropic.com
 
# 2. In Focus:
Settings → AI → Claude
Paste key → Save
 
# Default model:
claude-sonnet-4-6
Vision analysis included automatically. Keys stored locally, never transmitted anywhere except Anthropic's API.
OpenAI API Key
Reliable, fast. gpt-4o-mini is very affordable.
# 1. Get your key
platform.openai.com
 
# 2. In Focus:
Settings → AI → OpenAI
Paste key → Save
 
# Default model:
gpt-4o-mini
Vision analysis included automatically. Keys stored locally, never transmitted anywhere except OpenAI's API.
Privacy

Your data stays
on your Mac

Focus is built on a simple principle: the most sensitive productivity data imaginable should never leave your device.

What Focus does
✓Stores all data in a local SQLite database on your Mac
✓Reads active app name and window title via Accessibility API
✓Reads browser URL if you grant Automation permission
✓Takes periodic screenshots for vision analysis, then discards them immediately — only the text description is saved
✓Sends session data to your chosen AI provider (Ollama stays 100% local)
What Focus never does
✗Send any data to Focus servers — there are no Focus servers
✗Require an account or sign-in of any kind
✗Log keystrokes or clipboard content
✗Store screenshots anywhere on disk
✗Phone home, track usage, or collect analytics
Data lives at: ~/Library/Application Support/focus-session/focus-session.db
